# 📖 About the Project

The **AI Banking Assistant** is an intelligent chatbot developed using **Retrieval-Augmented Generation (RAG)**. The application retrieves relevant banking information from a **FAISS Vector Database** using **Ollama Embeddings**, then generates accurate and context-aware responses with **Ollama**.

Users can ask questions related to various banking services through a simple and interactive Flask web application.

# ⚙️ Working Process

1. Load banking documents.
2. Split documents into smaller chunks.
3. Generate embeddings using Ollama Embeddings.
4. Store vectors in the FAISS database.
5. User selects a banking service.
6. User enters a banking-related question.
7. LangChain retrieves the most relevant document chunks.
8. Ollama generates a context-aware response.
9. The response is displayed in the Flask web application.

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/AI-Banking-Assistant-RAG.git
```

### Navigate to Project

```bash
cd AI-Banking-Assistant-RAG
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Generate Embeddings

```bash
python embeddings.py
```

### Run the Application

```bash
python app.py
```

### Open Browser

```text
http://127.0.0.1:5000
```
